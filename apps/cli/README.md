# AgentX CLI

The AgentX CLI lets you log in, search, publish, review, and install AgentX artifacts from the terminal.

Primary command:

- `agentc`

Compatibility alias:

- `agentx`

Key commands:

- `agentc login`
- `agentc search`
- `agentc publish`
- `agentc reviews`
- `agentc review`
- `agentc install`

For full project documentation, see the repository root `README.md`.
